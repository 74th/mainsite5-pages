  • 日付は私がプレイした日付。
  • スコアは 5 点満点。☆5:G ドハマリ、☆4:楽しい、☆3:普通に楽しい、☆2:私になじまない点もあるが基本的に楽しい、☆1:not for me。

2017/07 Dead Cells

  • コンソール: PC(Steam)
  • ジャンル: ローグヴァニア
  • スコア: 5☆☆☆☆☆ ドハマリ
  • 難易度: 程よく難しい

メトロイドヴァニアの探索要素と、ローグ系アクションのアイテム取得していく感じが、良い感じにマッチしたゲーム。

遠距離弓プレイも、パリィアンドアタックプレイも、アタックアンドロールプレイも、ありとあらゆる 2D ゲームの戦闘が実現できる。

だから、どの武器が出てもハズレ感はなくて、その武器が出てもその武器での戦い方を考えて進める。この許されている戦略の幅が楽しくて、100 時間以上プレイしている。

難易度も、ノーマル → ハード → ベリーハード…と、敵の種類を強さが選べる。しかも、倒さずに逃げても構わないシステムになっている。探索も、基本ハズレはなくて、楽しく何度ももう一回とプレイしてしまう。

最初は赤中心で近接攻撃でバッサバッサ倒すの楽しい → まずはクリアを目指そうと、紫で、毒発生と毒+200%のアイテム（冷気、出血、炎のどれでも良い）がマッチするとボスが設置するだけで倒せることに気づいてクリア → パリィに快感を覚えて緑中心にする →…
